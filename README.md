Telegram Werewolf Bot

A feature-rich Telegram Werewolf game bot built with Python. The bot allows players to host multiplayer Werewolf games directly inside Telegram groups with automated game management, role assignment, voting, and interactive gameplay.

---

✨ Features

- 🎮 Multiplayer Werewolf gameplay
- 👥 Player lobby system
- 🎭 Automatic role assignment
- 🗳️ Voting system
- ⚖️ Win condition detection
- ⚡ Fast and asynchronous performance
- 🧩 Modular project structure
- 🔧 Easy to customize and extend

🚀 Installation

Clone the repository

git clone https://github.com/devanosvoldigold/Telegram-bot-.git
cd Telegram-bot-

Install dependencies

pip install -r requirements.txt

Configure the bot

Create a ".env" or update your configuration file:

BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN

---

▶️ Running the Bot

python main.py
